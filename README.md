# html-vocabulary

the words that html elements derives from

## The document element

| 元素   | 单词                      | 释义           | 描述                         |
| ------ | ------------------------- | -------------- | ---------------------------- |
| `html` | HyperText Markup Language | 超文本标记语言 | the root of an HTML document |

## Document metadata

| 元素    | 单词  | 释义 | 描述                                       |
| ------- | ----- | ---- | ------------------------------------------ |
| `head`  | head  |      | a collection of metadata for the Document. |
| `title` | title |      | the document’s title or name               |
| `base`  | cite  |      |                                            |
| `link`  | link  |      |                                            |
| `meta`  | meta  |      | document-level metadata                    |
| `style` | style |      | not represent content for the user.        |
